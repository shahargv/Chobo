# Chobo User Documentation

These guides are written for DBAs and operators who manage ClickHouse instances or clusters and need reliable backups, clear monitoring, and safe restores.

Chobo has both a web GUI and a CLI. The web GUI is usually easier for setup, review, and restore planning. The CLI is useful for automation, repeatable checks, and copying exact commands into runbooks.
